# lista-exercicios-python-poo
Este repositório contém a implementação de cinco sistemas desenvolvidos em Python utilizando conceitos avançados de Programação Orientada a Objetos (POO), incluindo:

Classes Abstratas (ABC)
Herança
Polimorfismo
Protocol
Duck Typing
Organização modular de projetos
Tecnologias Utilizadas
Python 3.12+
VS Code
Git
GitHub
Estrutura do Projeto

Cada questão foi organizada em uma pasta independente contendo:

classes do sistema;
arquivo principal main.py;
testes de execução;
comentários técnicos detalhados.
Como executar os projetos
1. Clonar o repositório

2. Acessar a pasta

3. Executar uma questão

Repita o processo para as demais questões.
